# Frontend – React App

Interface web do projeto Professor TI Onboarding, construída em React.

## Como Rodar o Projeto

### 1. Pré-requisitos
- Node.js (versão 16+)
- npm ou yarn

### 2. Configuração

Clone o repositório:
```bash
git clone https://github.com/seu-usuario/Professor-TI-Onboarding-Frontend.git
cd Professor-TI-Onboarding-Frontend
```

Instale as dependências:
```bash
npm install
```

Executar em modo de desenvolvimento:
```bash
npm start
```

Acesse em:
```bash
 http://localhost:3000
```

## Estrutura do Projeto
```
src/
├── components/     # Componentes ultilizados
├── img/          # Imagens usadas
├── Service/       # Comunicação com o backend
└── App.jsx          # Roteamento e estrutura principal
```
