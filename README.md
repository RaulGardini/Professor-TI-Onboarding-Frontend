# Frontend – React App

Interface web do projeto Professor TI Onboarding, construída em React.

## Como Rodar o Projeto
1. Pré-requisitos

Node.js (versão 16+)

npm ou yarn

2. Clonar o repositório
git clone https://github.com/seu-usuario/seu-projeto-frontend.git
cd seu-projeto-frontend

3. Instalar dependências
npm install

4. Executar em modo de desenvolvimento
npm start


Acesse em:
 http://localhost:3000

5. Gerar build de produção
npm run build


Os arquivos finais ficarão na pasta build/.

## Estrutura do Projeto
src/
├── components/     # Componentes reutilizáveis
├── pages/          # Páginas da aplicação
├── services/       # Comunicação com o backend
├── assets/         # Imagens e estilos
└── App.js          # Roteamento e estrutura principal

## Tecnologias Utilizadas

React (Create React App)

JavaScript

Axios (ou fetch)

HTML / CSS
